# QR Code Generator

A simple and responsive QR Code Generator built using HTML, CSS, and JavaScript.  
This project allows users to generate QR codes instantly from any text or URL and download them as images.

## Features

- Generate QR codes from text or links
- Multiple QR code sizes available
- Download generated QR code
- Responsive UI design
- Simple and beginner-friendly project

## Technologies Used

- HTML5
- CSS3
- JavaScript
- QRCode.js Library

## Project Structure

QR-Code-Generator/
│
├── index.html
├── style.css
├── script.js
└── README.md


## How to Use

1. Enter any text or URL in the input field
2. Select the QR code size
3. Click on the **Generate** button
4. Download the generated QR code using the **Download** button
